# 🌐 Full Stack Web Application

This is a full stack web application with user authentication and data storage.  
Users can sign up, log in, and interact with the system, with data securely stored in MongoDB.

---

## 🚀 Features
- User Signup & Login Authentication  
- Redirect to Home Page after login  
- Store user data in database  
- Simple and responsive UI  

---

## 🏗️ Tech Stack

### Frontend
- HTML5  
- CSS3  
- JavaScript  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB  

---

## 🔄 Application Workflow

### 1️⃣ Signup
- User creates a new account  
- Data is sent to backend  
- Stored securely in MongoDB  

### 2️⃣ Login
- User enters credentials  
- Backend verifies user data  
- On success → redirected to Home Page  

### 3️⃣ Home Page
- User can access application features  
- Data interactions happen through backend API  

### 4️⃣ Data Storage
- All user data is stored in MongoDB  
- Backend handles CRUD operations
